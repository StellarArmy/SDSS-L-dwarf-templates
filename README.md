SDSS L dwarf Templates
======================

This repo contains downloadable template spectra for L0-L8 dwarfs. The templates are described in detail by Schmidt et al. (2014). In short, each template contains the mean, median, error and spread of L dwarf spectra that have been corrected to zero radial velocity and normalized (at 8350--8450â„«).
